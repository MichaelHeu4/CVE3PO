Agent Install

Deployment guide with prefilled URLs for your current instance.

1. Download Installer

wget -O install.sh {{ install_script_url }}

2. Run Installation

sudo bash install.sh \
  --binary-url {{ binary_url }} \
  --sha256-url {{ sha256_url }} \
  --api-url {{ api_url }} \
  --token <AGENT_API_TOKEN>

The API token is shown in Modules → CVE3PO Agent.

3. Verify Service

sudo systemctl status cve3po-agent.timer --no-pager
sudo systemctl status cve3po-agent.service --no-pager
sudo journalctl -u cve3po-agent -n 100 --no-pager
